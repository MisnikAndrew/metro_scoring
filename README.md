# Скоринг станций Московского метро на статической карте <br>

Предоставленный код показывает топ лучших станций Московского метро (на основе близости к другим станциям с учётом пересадок и интервалов движения) <br><br>

Версия кода без конфигураций - pretty.cpp <br>
Стандартная версия - main.cpp <br><br>

Использование: <br>
get_best_stations(k), get_worst_stations(k) - получение k лучших и худших станций соответственно <br><br>

Список лучших станций - top.txt <br><br>
